# MusArt - Web Page Service Media (YouTube-like)

โปรเจคเว็บแบบ Static ที่จำลองหน้าใช้งานสไตล์แพลตฟอร์มมีเดียคล้าย YouTube โดยเน้นการแสดงผลวิดีโอ, รูปภาพ, และรายชื่อผู้สร้างคอนเทนต์จากข้อมูลในไฟล์ JavaScript

## ภาพรวมโปรเจค

MusArt ประกอบด้วยหลายหน้าเว็บที่เชื่อมต่อกันผ่านลิงก์และ query string เช่น หน้า feed วิดีโอ, หน้าดูวิดีโอ, หน้า photo feed, หน้าดูรูปเดี่ยว และหน้ารวมผู้สร้าง

ข้อมูลทั้งหมดถูกเก็บในไฟล์:

- `script/datascript.js`

โดยยังไม่ต้องใช้ฐานข้อมูลหรือ backend จึงสามารถเปิดใช้งานได้ทันทีผ่านเบราว์เซอร์

## ฟีเจอร์หลัก

- แสดงรายการวิดีโอแบบสุ่มบนหน้า Home (`index.html`)
- กดเข้าไปดูวิดีโอแต่ละรายการได้บนหน้า Watch (`watch.html`) ผ่านพารามิเตอร์ `id` และ `v`
- แสดงวิดีโอแนะนำเพิ่มเติมในหน้า Watch
- แสดงรายการรูปภาพแบบสุ่มบนหน้า Photo (`photo.html`)
- ดูรูปภาพแบบเต็มพร้อมข้อมูลเจ้าของบนหน้า Picture (`picture.html`) ผ่านพารามิเตอร์ `id` และ `p`
- แสดงรายชื่อผู้สร้างทั้งหมดบนหน้า Creaiter (`creaiter.html`)
- ใช้ไอคอนจากแพ็กเกจในโฟลเดอร์ `sup-pack/sub-icon`

## วิธีเริ่มใช้งาน

1. Clone หรือดาวน์โหลดโปรเจค
2. เปิดโฟลเดอร์โปรเจค
3. เปิดไฟล์ `index.html` ผ่านเบราว์เซอร์


## การปรับแต่งข้อมูล

แก้ข้อมูลได้ที่ `script/datascript.js` ในตัวแปร `dataweb`:

- `users[].PROFILE_IMG` : รูปโปรไฟล์ช่อง
- `users[].NAME_CH` : ชื่อช่อง
- `users[].VIDEO[]` : รายการวิดีโอ (`title`, `scr`, `view`, `date`)
- `users[].PHOTO[]` : รายการรูปภาพ

เมื่อต้องการเพิ่มคอนเทนต์:

1. วางไฟล์รูป/วิดีโอในโฟลเดอร์ `data/`
2. เพิ่ม path ของไฟล์ลงใน `datascript.js`
3. รีเฟรชหน้าเว็บ

## หมายเหตุ

- โปรเจคนี้เป็น Frontend ล้วน (HTML/CSS/JavaScript)
- ไม่มีระบบ login, upload, หรือฐานข้อมูลจริง
